# 💝 Интерактивная Валентинка на JavaFX
Красивое интерактивное приложение-открытка с анимированным сердечком, фотографиями и романтическим поздравлением. Идеальный подарок для особенного человека! ❤️

## ✨ Возможности

- 🫀 **Пульсирующее сердце** — SVG-анимация с эффектом биения
- 📸 **Плавное появление фотографий** — 12 фото появляются с эффектом fade-in
- 💌 **Романтический текст** — финальное поздравление с анимацией появления
- 🎨 **Красивый дизайн** — градиентный фон, современный интерфейс
- 🖱️ **Интерактивность** — кнопка запуска анимации

## 🛠️ Технологии

- **Java 17** — основной язык программирования
- **JavaFX 21** — графический интерфейс и анимации
- **Maven** — сборка проекта и управление зависимостями
- **SVG Path** — векторное изображение сердца
  
  ## 🚀 Как запустить

### Требования
- **JDK 17** или выше ([скачать](https://www.oracle.com/java/technologies/javase-downloads.html))
- **Maven** 3.6 или выше ([скачать](https://maven.apache.org/download.cgi))
- **(опционально)** JavaFX SDK, если будешь запускать без Maven

### Установка и запуск

1. **Клонируй репозиторий**
   ```bash
   git clone https://github.com/твой-username/valentine-card.git
   cd valentine-card
   
2. **Добавь свои фотографии**
- Положи 12 фотографий в папку src/main/resources/photos/
- Назови их 1.JPG, 2.JPG, 3.JPG, ... 12.JPG
- Формат должен быть JPG (можно переименовать любые фото)

3. **Запусти приложение**
    ```bash
    mvn clean javafx:run

## 🤝 Как использовать этот проект для своей валентинки

- Сделай fork этого репозитория (или просто скачай код)
- Замени фото в папке resources/photos/ на свои (обязательно 12 штук, формат JPG)
- Измени текст поздравления под свой случай
- При желании измени цвета под свой вкус
- Собери и запусти — готово! 🎁
